# 难点解决方案
## 运动跟随
鼠标决定蛇的运动方向
## 监听keydown的原因
不是按一下才走一下，可以按住方向键不停的走动。
按键不是为了让其走，而是为了改动蛇头的方向。
## 蛇可以使用数组的形式制作
  具体形式：[[3,1,'head'],[2,1,'body'],[1,1,'body']],
  第一个数组代表蛇头，3标识位置横坐标点，1标志位置纵坐标点
